### 🤔 타입스크립트란 ?

```
타입스크립트는 자바스크립트를 베이스로 하는 슈퍼셋업 자바스크립트이다. 자바스크립트를 한단계 감싸는 언어라고 볼 수 있다.
```

### ⭐ 대표적 특징

```
statically typed : 컴파일 시간에 타입에 대한 오류를 잡을 수 있다.
constructoerfunciton을 이용하면 클래스를 이용해 object를 만드는 것처럼 간편하게 object를 만들 수 있다.
```

### ✏️ 타입 스크립트 사용 이유

```
강력한 객체지향 프로그래밍이 가능하고, object-oriented-programming, modern programming 패러다임으로 이루어져 있다.

객체를 위주로 프로그래밍을 해나가는, 모듈성 있는 프로그래밍이다. 재사용성이 높고, 객체 단위로 확장해 나갈 수 있는 확장성도 놓고, 기존 코드의 문제 해결이나 근본을 쉽게 해결할 수 있는 유지 보수성도 굉장히 높다.
```

### 🖥️ 실행 방법

```
install : npm install -g ts-node
실행 명령어 : ts-node main.ts

tsc main.ts
➡️ 이런식으로 tsc 뒤에 파일명을 붙혀주면 js파일로 변환이 되고, 업데이트된 내용은 반영되지 않는다. 매번 컴파일러를 실행해줘야 한다.

watch mode : tsc -w
➡️ watch 모드를 이용하면 파일이 업데이트 될 때마다 계속 업데이트를 해준다.
```
